On9amer

View Github

On9amer is a web application that uses the Riot League of Legends API to request and display information about a person's previous 10 games of League of Legends played. Game data can be revealed by clicking an icon from the row of champion images.

  • PHP is used to fetch data from the API, while JQuery processes the data and creates the user interface
  • To test it out, type "turtlethecat" into the text box and push "Submit"
  • Created in June 2014, last updated November 2014

Relief Party

Github

Relief Party is a collection of mini games that are meant to portray situations that may arise during various different natural disasters.

  • This was a submission at the 12-hour International Women's Hackathon, hosted by the UW Women in Informatics group
  • Coded in JavaScript and uses the Phaser.io framework
  • Created November 2014

League of Tiles

Game+=Code

View

Game+=Code is a collaborative game community of ten second games coded in JavaScript that functions as motivation for new developers to create and upload their own games.

  • This is the first project I worked on outside of classwork after discovering I wanted to pursue computer science
  • It was conceptualized, created, and presented to a panel of judges in under 24 hours with a team of 5 for the hack-a-thon CodeDay
  • Created November 2013
