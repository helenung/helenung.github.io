the catvenger hunt

View Github

the catvenger hunt was created during nwhacks, western Canada's largest hackathon. It is a web app optimized for mobile use. Users are visited by cats as they move from location to location, and can play with the cats for as long as the cat stays. Cats leave currency, which users can then use to order more cats. The goal: fill the world with cats, or try to collect them all.

  • Created with a team of three in under 24 hours and demoed to the participants and judges of the hackathon
  • Incorporates the Google Maps API
  • Front-end/UI coded with HTML5, CSS3 (Bootstrap), JavaScript (jQuery, AngularJS)
  • Back-end consists of PHP, MySQL

The Bodemer Collection

View Github

The website for the Bodemer Collection, a collection of antique medical tools collected by the University of Washington's Department of Bioethics and Humanities. The research team has spent many years researching the individual items in the collection, and this website is meant to showcase those items. I took over responsibilities as the webmaster in September 2015. Since then, I have been responsible for maintaining the site, updating items on the web as the research team makes progress, and revamping the site visually.

  • Coded in HTML5, CSS3, and PHP
  • Utilizes MySQL, Bootstrap

Local-do.co

Local-do.co was created during the 2015 Dubhacks. It is a web application optimized for mobile-use that allows users to create, facilitate, and participate in scavenger hunts.

  • We used the Clarifai API which allows users to verify that they found the correct scavenger hunt item and earn points accordingly
  • Firebase powers the back-end, being our data storage solution as well as user authenticator
  • Coded in HTML5, CSS3, Javascript. Utilizes JQuery, Bootstrap, and Node.js
  • Created in 24 hours in October 2015

FindRhy.me

FindRhy.me is a location-based web application that allows users to submit poems to locations. Once submitted, other nearby users can view the original poem. FindRhy.me is a creative endeaver to allow people to feel a connection to the places they visit and gain an insight into the people that were there before. You can test it out now by submitting a poem with your name or alias!

  • Created with a team of four during the annual 24-hour hack-a-thon DubHacks, hosted at the University of Washington, Seattle
  • Incorporates the following APIs: Google Maps API, Zillow API, Bootstrap, Forecast.io
  • Coded in HTML5, CSS3, JavaScript, Angular.js, PHP, and MySQL
  • FindRhy.me was the recipient of the "Best NC.me branded domain name" at the 2014 DubHacks hack-a-thon
  • You can also find the submission post at ChallengePost here.
  • Created in 24 hours in November 2014

On9amer

View Github

On9amer is a web application that uses the Riot League of Legends API to request and display information about a person's previous 10 games of League of Legends played. Game data can be revealed by clicking an icon from the row of champion images.

  • PHP is used to fetch data from the API, while JQuery processes the data and creates the user interface
  • To test it out, type "turtlethecat" into the text box and push "Submit"
  • Created in June 2014, last updated November 2014
